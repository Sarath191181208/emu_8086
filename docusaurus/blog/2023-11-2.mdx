# Support for execution of `OR` ins 

## Execution of `OR` instructions
The `OR` instruction performs a bitwise OR operation on the value in register `rs1` and `rs2` and stores the result in register `rs1`. The syntax is:
```asm6502 showLineNumbers
OR AX, BX
```

## Compilation of `XOR` ins 

It is used to perform bitwise XOR operation on the value in register `rs1` and `rs2` and stores the result in register `rs1`. The syntax is:
```asm6502 showLineNumbers
XOR AX, BX
```

## Execution of the `xor` instruction
The `xor` instruction performs a bitwise XOR operation on the value in register `rs1` and `rs2` and stores the result in register `rs1`. The syntax is:
```asm6502 showLineNumbers
xor AX, BX
```
Algorithm: 
op1 <- op1 XOR op2
